#+TITLE: Merita Roadmap
#+AUTHOR: Paul H. McClelland
#+OPTIONS: toc:0 num:nil

Tentatively planned features for future releases.

* v0.5.1

- [ ] Filter presets for ~merita-browse~ and export: =biosketch= (peer-reviewed, last 5 years, significant authorship), =pipeline= (submitted + in-prep), =promotion= (all active entries)
- [ ] Author truncation: configurable threshold (default 6) with "et al." for citation formatters, following AMA 11th edition conventions

* v0.6.0

- [ ] Link visualization via Graphviz: ~merita-graph~ command emits =.dot= file, shells out to =dot -Tsvg=, displays in Emacs buffer; nodes positioned chronologically on x-axis, shaped by entry type, edges labeled by relation type; ~pipeline~ links rendered as thick directional arrows

* v0.7.0

- [ ] Journal impact factor lookup via OpenAlex source metadata; auto-populate ~impact_factor~ during DOI import or metrics update, with historical-vs-current distinction
- [ ] PubMed integration via NCBI E-utilities: PMID-based metadata fetch, MeSH term auto-population, PMC full-text link resolution
- [ ] Altmetric Attention Score via the free Details Page API; social media mentions, news coverage, policy document citations

* v0.8.0

- [ ] Multi-format CV templates: HTML and Markdown export alongside LaTeX; template selection per export or per entry-type override
- [ ] Grant reporting: publication lists filtered by grant period, formatted for NIH/NSF requirements, with citation metrics summaries for funded work

* Future

- [ ] Collaboration network: extract co-author relationships, compute collaboration frequency, display as adjacency list or report
- [ ] Promotion packet generator: combine publications, presentations, grants, awards, and service into a single document (LaTeX, Org, or Markdown)
- [ ] Embark actions for entry candidates in ~completing-read~: view, edit, delete, copy DOI, open URL, fetch metrics
- [ ] Org-mode integration: org-capture templates, org-agenda deadline tracking, export to structured headings with property drawers
- [ ] Syncing and portability: database checkpoint/close for Syncthing/Dropbox workflows, path normalization for cross-machine use
